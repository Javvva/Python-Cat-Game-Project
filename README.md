**# Python-Cat-Game-Project**
This game is a 2d platformer project using mainly the pygame library. This project was a final project for a advanced python university course and I decided to try and make a simple game that could be built upon. Though the code is messy, it was my first time in game development and doing a big coding project in general. All the assests used are linked here in my itch.io account [Intch.io Javvvvvva](https://itch.io/my-collections).

**Features:**
- Player physics
- Player movement
- Music & sounds effects
- Camera/Zoom functions
